## Project overview
A clean, well-structured ASP.NET Core backend demonstrating:
- Clean Architecture (Layers: API / Application / Domain / Infrastructure)
- RESTful APIs with Entity Framework Core
- SQL Server (or PostgreSQL) persistence
- Docker-ready for containerization
- Basic authentication (JWT) and unit tests

This repository is a minimal, production-minded starter to showcase architecture and coding practices for interview and portfolio purposes.

---

## Tech stack
- .NET 7+ / .NET 8 (update to latest installed)
- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / Azure SQL
- Docker (optional)
- xUnit (unit tests)
- GitHub Actions / Azure DevOps (CI pipeline - placeholder)
